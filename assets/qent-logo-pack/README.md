# Qent Logo Pack

## Contents

### Variants
| Variant | Use case |
|---------|----------|
| `icon` | Dark icon mark — for light backgrounds, app icons |
| `icon-light` | Light icon mark — for dark backgrounds |
| `full` | Icon + wordmark, dark text — for light backgrounds |
| `full-light` | Icon + wordmark, light text — for dark backgrounds |

### Sizes

**Icon (square)**
- 16x16 — favicon
- 32x32 — favicon retina
- 48x48 — small UI elements
- 64x64 — toolbar / nav
- 128x128 — app store small
- 256x256 — app store / social
- 512x512 — high-res app icon
- 1024x1024 — App Store / Play Store submission

**Full logo (3.33:1 ratio)**
- 200x60 — navbar / compact header
- 400x120 — standard usage
- 800x240 — hero sections / marketing
- 1600x480 — print / high-DPI

### Formats
- **SVG** — scalable, use for web and anywhere you need crisp rendering at any size
- **PNG** — rasterized at specific sizes with transparent backgrounds

## Brand Colors
- Primary dark: `#0D0D0D`
- Accent orange: `#E85D24`
- Light text: `#F5F5F5`

## Font Note
The SVGs use `Arial Black / Impact` as a system-safe fallback. For production use, 
consider converting text to outlines in Figma/Illustrator, or swapping to your brand 
font (the original concept used **Sora Bold 700**).
